# brutus

Wymagane zmienne środowiskowe dla pełnego raportu:

- `PERPLEXITY_API_KEY`
- `FIRECRAWL_API_KEY`
- `GEMINI_API_KEY` albo `GOOGLE_GENERATIVE_AI_API_KEY`

Opcjonalnie:

- `GEMINI_MODEL` - domyślnie `gemini-3.1-flash-lite-preview`
- `GEMINI_API_BASE_URL` - domyślnie `https://generativelanguage.googleapis.com/v1beta`
- `KRS_OPEN_API_URL` - domyślnie `https://api-krs.ms.gov.pl/`
